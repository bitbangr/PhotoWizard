Blending Two Images

Begin with an image already loaded into PhotoWizard.

Load Second Image

 

  1. First select Tools

  2. Next select Blend Images

  3. Then from Blend Image toolbar select the second image.

  4. You now have two images to work with. You can re-size, rotate, and reposition this second image with the handles

 

Adjust Image Settings

  1. Select the triangle with three Dots to bring up the following four blending options.

    Blend Options Description
    Reset Discard all changes to imported image.
    Scale Fit Scale the imported image to fit the size of the first image.
    Swap Swap the foreground and background images.
    Aspect Lock Lock the aspect ratio of the imported image.

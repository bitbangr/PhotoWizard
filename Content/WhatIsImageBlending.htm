What is Image Blending?

Image blending allows you to combine two images into a new image using various tools and techniques.

Below are the key features and steps:

 

Key Features

  • Combine Images: You can combine two images using this tool.

  • Touch Gestures: Use touch gestures to move and rotate the foreground image.

  • Fit Images: Adjust the foreground image to fit the background image from settings.

  • Blending Modes: Combine images using different blending modes.

  • Adjust Transparency: Modify the transparency of the foreground image.

Using Masks

  • Transparent Masks: If you have masked the image already, the masked portion will be transparent if chosen as the foreground image.

  • Superimpose Images: Use masks to superimpose part of an image on top of another.

     

    By utilizing these features, you can create a seamless integration of multiple images into a single composition.

Adding Text to Images

  1. Start with an image loaded into PhotoWizard.

  1. First select Tools.

  2. Then select Add Text.

  3. Your image is now displayed with default text inside of a box with manipulation handles.

Resize Reposition Text

  1. Touch-drag the box handles to re-size and rotate the text.

    Two-finger drag inside the box to reposition the text.

Edit Text

  1. Select 'T' in the top menu bar or double tap text in box to open the text edit window.

    Update your text.

    Choose left, centre, or right alignment.

    Select check mark to keep changes or 'x' to discard.

Edit Text Properties

  1. Select Gear Icon in top menu to open text edit properties window.

    You can change the following:

    • Text Colour

    • Text Font

    • Shadow

    • Opacity

    Select the Text Font to bring up a font selection window.

Chose Blending Mode and Opacity

  1. Now touch the top menu Gear Icon to bring up the blending modes and opacity dialogue.

  2. Experiment with various blending modes and opacity.

  3. When you are happy with your blended image touch the check mark in the top menu bar to keep your results.

 

NOTES:

You have creative control over which parts of your images are combined by masking your background image.

 

You have an unlimited undo action so feel free to explore different combinations

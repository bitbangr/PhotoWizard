Adding Text to Images

  1. Start with an image loaded into PhotoWizard.

  1. First, select Tools.

  2. Then select Add Text.

  3. Your image is now displayed with default text inside a box with manipulation handles.

Resize and Reposition Text

  1. Touch-drag the box handles to resize and rotate the text.

    Two-finger drag inside the box to reposition the text.

Edit Text

  1. Select 'T' in the top menu bar or double-tap the text box to open the text edit window.

    Update your text.

    Choose left, centre, or right alignment.

    Select check mark to keep changes or 'x' to discard.

Edit Text Properties

  1. Select the gear icon in the top menu to open the text properties window.

    You can change the following:

    • Text Colour

    • Text Font

    • Shadow

    • Opacity

    Select Text Font to bring up a font selection window.

Save Your Edits

  1. When you're happy with your text edits, touch the check mark in the top menu to keep your results.

Congratulations! You have successfully added text to your photo!

 

NOTES:

 

You have unlimited undo actions, so experiment with different text options.
